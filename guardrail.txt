### 3D ASSET LOADING PROTOCOL
- **GLTF/GLB files**: Load via `useGLTF` from `@react-three/drei`. Place files in `/public/assets/` and reference by path (e.g., `/assets/model.glb`).
- **OBJ files**: Load via `useLoader(OBJLoader)` from `@react-three/fiber`. Same path convention.
- **Sketchfab models**: Embed via Sketchfab's `<iframe>` embed API using the model UID from extracted data. Respect Sketchfab's embed terms.
- **Fallback**: If no 3D asset is provided, render the default ambient particle field (minimum 1500 particles, inSphere distribution, scroll-linked parallax).
- **Performance**: All 3D models must be under 5MB. Use Draco compression for GLTF files. Lazy-load canvases with `Suspense` boundaries.

### GSAP ANIMATION PROTOCOL
- **ScrollTrigger**: Every chapter must have at least one ScrollTrigger-based reveal animation. Trigger points: `start: 'top 75%'`, `end: 'bottom top'`.
- **Staggered reveals**: Text nodes, cards, and imagery layers must fade/slide in with 0.1–0.2s stagger delays. No elements appear simultaneously.
- **Micro-interactions**: All buttons and interactive elements must have GSAP-powered hover states (scale, color shift, underline slide). No CSS-only hover effects.
- **Timeline sequencing**: Hero entrance must use a master `gsap.timeline()` with choreographed delays. Subsequent sections use independent ScrollTrigger contexts.
- **Cleanup**: All GSAP animations and ScrollTriggers must be killed on component unmount to prevent memory leaks.
